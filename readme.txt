Easily integrate UserDeck's embedded support software into your WordPress site.

== Description ==

[UserDeck](http://userdeck.com) provides support software such as knowledge base and feedback widgets to embed into existing websites.

Gather feedback from your users which integrates with knowledge base articles to reduce support time.

Add a knowledge base with UserDeck easily to any page of your WordPress site.

This plugin generates the javascript embed code for the feedback widget to the footer of your WordPress automatically. To turn a page into a knowledge base simply add the short code.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the userdeck folder to your wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to the settings page.
4. Enter your UserDeck helpdesk ID.
5. To add the knowledge base to a page, simply add the short code '[usedeck_kb].
